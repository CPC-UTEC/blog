Classes

Each class lasts two hours. Classes have a theorical and practical part. The theorical part consists of a lecture-style class. The practical part is a set of problems in a contest using vjudge. The solutions of the problems are posted when the contest finishes. Moreover, each class have some recommended readings. Furthermore, from Monday to Friday there is a daily problem to practice, you can find them here.

The classes are on Monday and Wednesday in UTEC at A801 from 17:00 to 19:00.
